Faça um programa que leia dois números inteiros e verifique se eles são primos entre si. Em seguida, imprima uma mensagem informando se os números são primos entre si ou não.


### Exemplo 01

#### Entrada

    8 21

#### Saída

    São Primos entre SI.

### Exemplo 02

#### Entrada

    2 10

#### Saída

    NÃO São Primos entre SI.
