Faça um programa que leia um número inteiro e verifique se ele é positivo, negativo ou zero. Em seguida, imprima uma mensagem informando qual é o tipo do número.


### Exemplo 01

#### Entrada

    7

#### Saída

    POSITIVO

### Exemplo 02

#### Entrada

    -4

#### Saída

    NEGATIVO

### Exemplo 03

#### Entrada

    0

#### Saída

    ZERO
